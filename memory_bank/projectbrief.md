# Project Brief

## Goals

Создать прототип action roguelike на Godot 4 с главным меню, одной игровой сценой и базовыми боевыми механиками, который можно расширять до MVP из `PRD.md`.

## Scope

- Текущий подтвержденный scope берется из фактического состояния кода и высокоуровневого описания в `docs/README.md`.
- Канонический процент выполнения проекта определяется только по таблице deliverables ниже.
