# Product Context

## Product Intent

GunBlade задуман как средневековый top-down roguelike с RPG-элементами. В актуальном состоянии репозиторий содержит ранний прототип, который подтверждает базовый игровой цикл: вход из меню, загрузка уровня, бой с противником и минимальное сохранение состояния run.

## User Value

- Игрок может быстро запустить прототип из главного меню.
- Игрок может перемещаться по уровню, атаковать и взаимодействовать с базовым врагом через боевую петлю.
- Разработчик уже имеет основу для дальнейшего расширения в сторону MVP из `PRD.md`.

## Confirmed Product Boundaries

- Подтверждено кодом: single-scene prototype, one player archetype, one basic enemy archetype, simple dungeon scene.
- Не подтверждено кодом: полноценный progression loop, многоэтажное подземелье, расширенный UI, системы предметов и мета-прогресс.
