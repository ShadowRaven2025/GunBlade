# Active Context

## Current Focus

Расширение игрового прототипа: добавлены базовое окружение арены, полоски здоровья, статичный враг с контактным уроном и респавном после смерти.

## Active Decisions

- Источником высокоуровневой архитектуры остается `docs/README.md`.
- В `scripts/core/Player.gd` количество кадров теперь задается отдельно для `idle`, `run` и `attack` и синхронизируется со сменой текстуры.
- `cut_ui_sprites.ps1` не связан с багом персонажа: он режет только UI-ассеты, а проблема была в runtime-настройке `Sprite2D`.
- Атака переведена в отдельное состояние `is_attacking`, чтобы `idle/run` не перехватывали спрайт до завершения attack window.
- Slash-эффект реализован минимально через runtime `Line2D`, чтобы не зависеть от наличия отдельного FX-ассета в репозитории.
- Враг переведен на простую статичную модель поведения: idle-анимация, урон по касанию, смерть и автоматический респавн по таймеру.
- Полоски здоровья реализованы непосредственно в сценах через `ProgressBar`, без отдельной логики shared-компонента.
