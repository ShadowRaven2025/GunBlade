# Medieval Roguelike RPG - Development Plan

## Phase 1: Prototype (Week 1-3)

### Week 1: Project Setup
- [ ] Настроить Godot 4.x проект
- [ ] Настроить систему контроля версий (GDScript игнорировать)
- [ ] Создать базовую структуру директорий
- [ ] Настроить импорт ассетов (Tiny Swords)
- [ ] Создать тестовую сцену с тайлсетом

### Week 2: Player Movement
- [ ] Реализовать персонажа (спрайт из Tiny Swords)
- [ ] Добавить движение по WASD/стрелкам
- [ ] Реализовать изометрическое движение
- [ ] Добавить базовую анимацию (Idle, Run)
- [ ] Настроить камеру (следование за игроком)

### Week 3: Basic Combat
- [ ] Добавить систему атаки (авто-атака по клику/касанию)
- [ ] Создать тестового врага (простой спрайт)
- [ ] Реализовать столкновение атака/получение урона
- [ ] Добавить HP персонажа и врагов
- [ ] Реализовать смерть врага и игрока (тестовая)

## Phase 2: Core Gameplay (Week 4-9)

### Week 4-5: Character Classes

### Week 6-7: Dungeon Generation

- [ ] Добавить типы комнат (обычная, сундук, босс)
- [ ] Реализовать переход между этажами (лестница)
- [ ] Добавить 5 этажей подземелья

### Week 8-9: Inventory & Equipment
- [ ] Создать систему инвентаря
- [ ] Реализовать экипировку (оружие, броня)
- [ ] Добавить систему предметов (拾取, использование)
- [ ] Создать UI инвентаря
- [ ] Добавить систему слотов экипировки

## Технические Задачи

### GDScript Structure
```
scripts/
├── core/
│   ├── Game.gd              # Глобальное состояние
│   ├── Player.gd            # Базовый игрок
│   ├── Enemy.gd             # Базовый враг
│   └── Camera.gd            # Камера
├── characters/
│   ├── Warrior.gd
│   ├── Mage.gd
│   └── Rogue.gd
├── combat/
│   ├── Attack.gd            # Система атаки
│   ├── Damage.gd            # Расчёт урона
│   └── StatusEffect.gd      # Эффекты
├── systems/
│   ├── Inventory.gd
│   ├── Equipment.gd
│   ├── DungeonGenerator.gd
│   ├── Alchemy.gd
│   └── Roguelike.gd         # Runes, Curses, Relics
└── ui/
    ├── MainMenu.gd
    ├── GameHUD.gd
    └── InventoryUI.gd
```
